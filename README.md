# Goodwall Landing Page

A modern, responsive landing page for Goodwall built with plain HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - page markup and content
- `styles.css` - full site styling and responsive layout rules
- `script.js` - interactivity (menu toggle, scroll effects, counters, reveal animations, smooth scrolling, dynamic footer year)

## Features

- Responsive design for desktop, tablet, and mobile
- Animated hero and section reveal effects
- Count-up statistics on scroll
- Sticky navigation with mobile drawer menu
- Dynamic footer year using JavaScript `Date` object

## How to Run

1. Open `index.html` directly in your browser, or
2. Use a local server (recommended), such as VS Code Live Server.

## Customization

- Update content in `index.html`
- Change colors, spacing, and typography variables in `styles.css` under `:root`
- Edit behaviors and animations in `script.js`

## Notes

- External dependencies:
  - Google Fonts (`Syne`, `DM Sans`)
  - Font Awesome CDN for icons
